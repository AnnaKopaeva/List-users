# [List-users](https://annakopaeva.github.io/List-users/build/index.html)

```
    $ git clone https://github.com/AnnaKopaeva/List-users.git
    $ cd List-users
    $ npm install
    $ npm build
    $ npm start
```
Project is running at http://localhost:9000/
